# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Two-player gameplay (Player X vs Player O)
- Real-time win detection
- Draw condition handling
- Interactive and responsive UI
- Reset / Restart game functionality

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (DOM Manipulation)

## 📂 Project Structure
/tic-tac-toe
│── index.html
│── style.css
│── script.js
## 🎯 How to Play
1. Player X starts the game
2. Players take turns marking cells
3. First player to align 3 marks (row, column, or diagonal) wins
4. If all cells are filled without a winner, it's a draw
